API Response Properties

currencylayer API results are delivered in JSON format for maximum usability. Find below descriptions for the JSON properties returned by the API:

"success" true if your query succeeds, and false, if not. (see API Error Codes) "terms" All data and services provided are for informational purposes only and all usage is at your own risk. "privacy" Legal reminder for the user to take a look at our Privacy Policy. "timestamp" Exact date and time (UNIX) the exchange rates were collected. "source" The currency to which all exchange rates are relative. (default: USD) "quotes" Contains all Exchange Rate values, consisting of the currency pairs and their respective Conversion rates.
